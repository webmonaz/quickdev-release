# QuickDev

Desktop utility for developers to manage terminal presets, Docker containers, network ports, and local services.

## Features

- **Terminal Presets** - Save and manage command templates
- **Profiles** - Workspace-scoped presets and terminal selection
- **Hosts Manager** - Safely edit /etc/hosts with backup/restore
- **mDNS Discovery** - Auto-register local services for mobile app testing
- **Skills Sync** - Sync skills across AI tools and projects
- **AI Tools Hub** - Discover AI tool configs and assets
- **Port Monitor** - List and filter active ports with process info
- **Auto-Update** - Built-in updates via GitHub Releases
